# Spaced Repetition

Spaced repetition is a learning technique where review intervals grow over
time. Instead of cramming, you revisit a concept right before you are likely
to forget it.

The core idea comes from the **forgetting curve** (Ebbinghaus): memory decays
exponentially, but each successful recall flattens the curve, so the next
review can be scheduled further out.

Common algorithms: SM-2 (Anki), Leitner boxes, FSRS.
